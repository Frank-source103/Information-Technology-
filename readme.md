# Department Website

A modern department website built with Flask, HTML, CSS, JavaScript, and SQLite.

## Features
- Editable pages (Home, About, Info)
- Admin Dashboard
- Contact Form storing messages
- SQLite Database
- Modern responsive UI

## Installation

1. Clone project
2. Install dependencies
   pip install -r requirements.txt
3. Run app
   python app.py
4. Open browser
   http://127.0.0.1:5000
